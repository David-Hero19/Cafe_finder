# WorkBrew — Remote Cafe Finder

Find cafes that actually work. Filter by WiFi speed, noise level, and power outlets across 6 cities.

## Stack
- HTML / CSS / Vanilla JS
- Python serverless API (Vercel)

## Run Locally
```bash
python -m http.server 3000
```
Open http://localhost:3000

## Deploy
Import repo to [vercel.com](https://vercel.com) and click Deploy.
